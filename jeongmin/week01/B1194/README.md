# BOJ 1194 - 달이 차오른다, 가자.

## 📌 문제 설명

열쇠 6종류와 그 열쇠에 맞는 문이 존재하는 미로 탈출 최소 거리 문제

## 💡 해결 아이디어

1. 최소 거리니까 bfs로 풀어야겠다.
2. 열쇠를 주웠을 경우 왔던 경로로 되돌아가야 하는 경우도 생기겠는데?
3. 열쇠를 주울 때마다 bfs를 다시 호출해볼까?
4. 메모리 초과가 나네. 열쇠를 주웠다는 걸 다르게 어떻게 표기하지?
5. 열쇠의 종류가 적으니 비트마스킹 활용, visited를 3차원으로 확장하여 열쇠의 여부를 표기

## 🧠 코드 해설

```java
boolean[][][] visited = new boolean[n][m][64];
```

visited를 3차원 배열로 만들어서 열쇠 보유 여부를 반영

```java
				if (keys.contains(map[nx][ny])) {
					int index = map[nx][ny]-'a';
					if ((k & (1 << index)) == 0) k = k ^ (1 << index);
				}
```

열쇠가 없을 경우 열쇠를 습득

```java
				visited[nx][ny][k] = true;
				q.add(new int[] {nx, ny, t+1, k});
```

visited에 열쇠 정보 반영  
보유중인 열쇠를 queue에 넣어 기록 유지

## 🚀 느낀점

열쇠 정보를 저장해야 된다고는 생각했는데 visited를 3차원으로 팔 생각을 못해서 좀 어렵게 돌아간 것 같다.
