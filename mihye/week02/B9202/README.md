## 📝 문제 요약

4*4크기의 그리드에서 최대한 많은 단어를 찾는 게임 Boggle

Boggle에서는 단어를 가로,세로,대각선을 이용해서 만들 수 있다 (단 한주사위는 단어 한번만 사용)

1글자, 2글자로 이루어진 단어는 0점, 3글자, 4글자는 1점, 5글자는 2점, 6글자는 3점, 7글자는 5점, 8글자는 11점이다. 점수는 자신이 찾은 단어에 해당하는 점수의 총 합이다.

단어 사전에 등재되어 있는 단어의 목록과 Boggle 게임 보드가 주어졌을 때, 얻을 수 있는 최대 점수, 가장 긴 단어, 찾은 단어의 수를 구하는 프로그램을 작성하시오.

## 💡 접근 방법

- **알고리즘/자료구조: DFS**
- **핵심 아이디어**
    - 전체 맵을 탐색하면서 단어사전에서 등재된 단어를 완성하는지 탐색

## ✅ 풀이 과정

1. 모든 맵의 좌표를 탐색
2. 해당 좌표(시작점)이 사전 단어의 첫 글자면 dfs 실행
3. DFS 탐색 시 target 문자열과 현재 찾으려하는 글자 인덱스를 함께 넘겨주고
    1. 8방향으로 나아가면서 그 방향 글자가 타켓글자와 같으면 인덱스 +1 하여 다시 탐색 반복
4. 타켓글자의 마지막 문자까지 탐색을 완료했으면 점수 계산 및 가장 긴 단어 업데이트
