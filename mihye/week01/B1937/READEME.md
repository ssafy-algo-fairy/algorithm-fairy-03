## 📝 문제 요약

상,하,좌,우로 맵을 움직일때 현재 값보다 더 큰 값으로만 이동한다.

이때 어느 좌표에 놓아야 최대한 많은 칸을 이동할 수 있는지 궁금할 때 경로 최대값

## 💡 접근 방법

- **알고리즘/자료구조: DFS, DP**
- **핵심 아이디어: DP로 해당 위치에서 나아갈수있는 경로의 수를 저장하며 깊이우선탐색을 진행**

## ✅ 풀이 과정

### 해결 전략 수립

- 깊이 우선 탐색하면서 경로 찾기
- 4방향으로 해당값이 현재값보다 큰 것만 탐색
    - 만약 가려고하는 방향의 dp가 저장되어있지않다면 dfs 탐색
    - 가려는 방향의 dp가 저장되어있다면 현재 dp를 가려는 방향 dp+1로 업데이트(4방향중 가장 큰값)
- 4방향 탐색을 종료했는데도 dp가 업데이트되지않다면 1으로 저장.
- 현재 dp를 기준으로 정답 max 업데이트

## 📌 배운 점 / 주의사항

### 시간초과

- DFS로만 최대값 구하기 → 시간초과
- 시작점기준으로 DP 저장 → 시간초과

⇒ DFS 탐색하면서 해당 좌표의 나아갈수있는 경로 수 저장

### 욕심쟁이 판다

⇒ 욕심쟁이 팬다

⇒ 욕심을 부리지말자.
