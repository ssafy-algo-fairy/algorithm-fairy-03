### 💡 접근 방식

> 트리에서 리프 노드(차수가 1인 정점)의 개수를 세고,  
> 이를 이용해 필요한 최소 개수를 계산하는 문제입니다.

트리의 끝점인 **리프 노드들끼리 짝을 지어 연결**한다고 생각하면,

- 리프가 1개면 `1`
- 리프가 2개면 `1`
- 리프가 3개면 `2`
- 리프가 4개면 `2`


```java
(리프 개수 + 1) / 2
```

---

### 🔹 1단계 – 트리 입력 받기

- 정점 수 `N`을 입력받고
- 인접 리스트로 트리를 저장

```java
graph = new ArrayList[N];
for (int i = 0; i < N; i++) {
    graph[i] = new ArrayList<>();
}
```

---

### 🔹 2단계 – 리프 노드 개수 세기

트리에서 리프 노드는 **차수가 1인 정점**

```java
int cnt = 0;
for (int i = 0; i < N; i++) {
    if (graph[i].size() == 1) {
        cnt++;
    }
}
```

여기서 `cnt`는 리프 노드의 개수

---

### 🔹 3단계 – 정답 계산

리프 두 개를 하나의 경로로 처리할 수 있으므로,  
필요한 최소 개수는 리프 개수를 2개씩 묶었을 때의 개수와 같음

```java
(cnt + 1) / 2
```

---

### 💻 핵심 코드

```java
int cnt = 0;

for (int i = 0; i < N; i++) {
    if (graph[i].size() == 1) {
        cnt++;
    }
}

System.out.println((cnt + 1) / 2);
```

---

### ⏳ 복잡도 분석

- **시간 복잡도:** `O(N)`
    - 인접 리스트 구성: `O(N)`
    - 리프 노드 개수 세기: `O(N)`

- **공간 복잡도:** `O(N)`
    - 인접 리스트 저장

---

### ⚠️ 구현 포인트

- 트리이므로 간선 수는 항상 `N-1`
- 리프 노드는 차수가 1인 정점
- 리프를 두 개씩 묶는다고 생각하면 `(cnt + 1) / 2` 공식을 바로 적용할 수 있음
